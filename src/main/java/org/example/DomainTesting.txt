Условие существования треугольника: сумма любых двух сторон должна быть больше третьей.
Возможные типы треугольников:
Равносторонний: все стороны равны.
Равнобедренный: две стороны равны, третья — нет.
Разносторонний: все стороны различны.
Пример таблицы тестов:

  Сторона A	 Сторона B	Сторона C	Ожидаемый результат

    3	       3	       3	    Равносторонний треугольник
    3	       3	       4	    Равнобедренный треугольник
    3	       4	       5	    Разносторонний треугольник
    1	       2	       3	    Не является треугольником
    0	       0	       0	    Не является треугольником
